# violoncello-kai-os
Data-saving, performant text-only web browser for Kai OS devices (v2.5-4)

Features:
- 5 themes: light, dark, sepia, blue, console
- Intuitive navigation system for scrolling/links
- Consistent UI that fits the vibe of Kai OS
- No ads
- Bookmarking system
- Lightning quick!

Why use Violoncello?
- Less data usage (loading videos, images, ads, etc.)
- Quicker
- Fewer distractions
- Great for reading articles!
